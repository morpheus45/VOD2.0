PIPSILY sur iPhone / iPad

Tu peux installer PIPSILY comme une application Apple (PWA) directement depuis Safari.

  1. Ouvre https://morpheus45.github.io/VOD/ avec Safari.
  2. Touche le bouton Partager (carré avec flèche).
  3. Choisis Sur l’écran d’accueil.
  4. Valide Ajouter.

Ensuite, lance PIPSILY depuis l’écran d’accueil comme une app.

Compatibilité

Fonctionne sur iOS / iPadOS récents via Safari. Pas besoin d’App Store pour cette version web app.
