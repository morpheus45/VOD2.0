Installer PIPSILY

Choisissez votre appareil et suivez les étapes — moins de 5 minutes.

iPhone & iPad
Ajoutez PIPSILY à votre écran d'accueil — comme une vraie app, sans passer par l'App Store.
✓ iOS 14+
✓ iPhone SE / 8 / X / 11–16 Pro Max
✓ iPad (toutes générations)
1
Ouvrez Safari — pas Chrome
Cette procédure fonctionne uniquement avec Safari. Si vous utilisez Chrome ou Firefox, ouvrez Safari d'abord.
2
Accédez à PIPSILY
Dans la barre d'adresse, tapez :
morpheus45.github.io/VOD
Attendez le chargement complet de la page.
3
Appuyez sur l'icône Partager 📤
En bas de l'écran (iPhone) ou en haut (iPad) — le carré avec une flèche vers le haut. Si vous ne la voyez pas, faites défiler la page vers le haut.
4
Appuyez sur « Sur l'écran d'accueil »
Faites défiler le menu vers le haut jusqu'à voir « Sur l'écran d'accueil » (icône carré avec +). Appuyez dessus.
Vérifiez le nom PIPSILY puis appuyez sur Ajouter en haut à droite.
5
C'est installé !
PIPSILY apparaît sur votre écran d'accueil. Appuyez dessus pour l'ouvrir en plein écran, sans barre Safari — comme une app native.
💡 Mise à jour automatique : L'app se met à jour silencieusement à chaque ouverture.
⚠ Page blanche ? Vérifiez votre connexion Wi-Fi et reconnectez-vous. Fonctionne aussi en 4G/5G.
📱
Android (téléphone & tablette)
Installez via Chrome (sans téléchargement) ou téléchargez l'APK natif pour une expérience optimale.
✓ Android 8+
✓ Samsung Galaxy
✓ Xiaomi / POCO
✓ Google Pixel
✓ OnePlus / Oppo / Sony
🌐 Via Chrome (PWA)
📦 APK natif
1
Ouvrez Chrome
Ouvrez Google Chrome sur votre téléphone Android.
2
Accédez à PIPSILY
Tapez dans la barre d'adresse :
morpheus45.github.io/VOD
3
Installez l'application
Une bannière « Installer PIPSILY » apparaît en bas — appuyez dessus.
Si absente : ⋮ 3 points → Ajouter à l'écran d'accueil.
4
Confirmez et profitez
Appuyez sur Installer. PIPSILY s'ajoute à votre écran d'accueil et au tiroir d'applications.
💡 Compatible : Samsung Galaxy, Xiaomi, Oppo, OnePlus, Google Pixel, Sony, Motorola et tous les Android avec Chrome.
1
Téléchargez l'APK PIPSILY
Appuyez sur le bouton ci-dessous pour télécharger l'application (~8 Mo).
2
Autorisez les sources inconnues (si demandé)
Si Android le demande : Paramètres → Installer des apps inconnues → activez pour Chrome → revenez et relancez.
3
Installez et ouvrez
Appuyez sur Installer. PIPSILY est maintenant dans vos applications.
⬇ Télécharger l'APK (v24)

Android TV & Google TV
APK natif optimisé pour la télécommande — navigation fluide avec les touches directionnelles.
✓ Sony Bravia Android TV
✓ Philips Android TV
✓ TCL / Hisense
✓ Chromecast with Google TV
✓ Nvidia Shield
✓ Xiaomi Mi Box / TV Stick
📦 APK via Downloader (recommandé)
🌐 Via navigateur
🔧 Via ADB (PC)
1
Activez les sources inconnues sur la TV
Paramètres ⚙️ → Sécurité et restrictions → Sources inconnues → activez pour Downloader.

Sur certaines TV : Paramètres → Applications → Accès spécial → Installer des apps inconnues.
2
Installez « Downloader » depuis le Play Store
Ouvrez le Play Store sur votre TV → cherchez Downloader (icône orange, par AFTVnews) → Installez. Gratuit.
3
Ouvrez Downloader et entrez l'URL
Lancez Downloader → dans le champ URL, tapez :
bit.ly/pipsily-apk
Appuyez sur Go. Le téléchargement démarre (~8 Mo).
4
Installez l'APK
Une fois terminé, appuyez sur Installer. Si un avertissement apparaît, appuyez sur Installer quand même.
5
Lancez PIPSILY et connectez-vous
Appuyez sur Ouvrir ou retrouvez PIPSILY dans Applications → Voir toutes. Connectez-vous avec votre compte.
6
Épinglez PIPSILY en tête de liste (optionnel)
Maintenez la sélection sur PIPSILY → Déplacer en tête de liste pour l'avoir toujours visible sur votre écran d'accueil TV.
✓ URL alternative complète :
github.com/morpheus45/VOD/releases/download/v24/PIPSILY.apk
💡 Mise à jour : Quand une nouvelle version est disponible, une notification apparaît dans PIPSILY. Répétez les étapes 3–4 pour mettre à jour.
1
Installez un navigateur depuis le Play Store
Sur votre TV, installez l'un de ces navigateurs :

• TV Bro — le plus adapté aux télécommandes (recommandé)
• Firefox for Android TV
• Chrome (si disponible sur votre TV)
2
Accédez à PIPSILY
Ouvrez le navigateur → saisissez :
morpheus45.github.io/VOD
Astuce : connectez un clavier Bluetooth pour taper plus facilement.
3
Connectez-vous et ajoutez en favori
Entrez vos identifiants. Ajoutez en favori pour un accès direct la prochaine fois.
💡 TV Bro est le navigateur le plus adapté aux télécommandes — navigation fluide avec les flèches directionnelles.
1
Activez le débogage ADB sur la TV
Paramètres → À propos → Construire → appuyez 7 fois → Options développeur → activez Débogage ADB.
2
Téléchargez l'APK sur votre PC
github.com/morpheus45/VOD/releases/download/v24/PIPSILY.apk
3
Installez via ADB
Dans un terminal PC (remplacez par l'IP de votre TV) :
adb connect <IP-TV>:5555
adb install PIPSILY.apk
⚠ Méthode avancée. Réservée aux utilisateurs à l'aise avec les outils Android. Préférez la méthode Downloader.

Samsung Smart TV
Installation du package .wgt Tizen. Modèles 2016 et ultérieurs. Nécessite un PC/Mac sur le même réseau Wi-Fi.
✓ Samsung 2016+ (Tizen 3.0+)
✓ Séries QLED / Neo QLED
✓ Séries 4K UHD 2016–2025
✓ The Frame / The Serif
🖥 Installateur auto (recommandé)
🔧 Manuel (développeurs)
1
Activez le Mode Développeur sur la TV
Sur la télécommande Samsung, appuyez sur Home / Menu
→ Paramètres ⚙️ → Assistance → À propos de ce téléviseur
→ Appuyez 5 fois rapidement sur OK (ou entrez le code 12345 si une boîte de dialogue s'ouvre)
→ Mode développeur → ON
→ Entrez l'adresse IP de votre PC (Windows : ouvrez cmd → tapez ipconfig → notez l'IP)
→ Redémarrez la TV (débranchez 10 secondes).
2
Téléchargez le package d'installation sur votre PC
Sur votre PC, rendez-vous sur :
morpheus45.github.io/VOD/samsung-tv.html
et téléchargez le ZIP d'installation PIPSILY Samsung.
3
Lancez l'installateur
Extrayez le ZIP et lancez :
• Windows : double-cliquez INSTALLER.bat
• Mac : ouvrez Terminal dans le dossier → bash install.sh

L'installateur détecte votre TV automatiquement, envoie le .wgt et installe PIPSILY.
4
Ouvrez PIPSILY sur la TV
Sur la TV → Apps → Mes applications. PIPSILY apparaît dans la liste. Lancez-le et connectez-vous.
📺 Guide complet Samsung TV ⬇ .wgt direct
💡 Même réseau requis : PC et TV Samsung doivent être sur le même Wi-Fi ou Ethernet.
⚠ Mode développeur : Se désactive après quelques jours sur certains modèles. Si l'app disparaît, relancez simplement l'installateur.
1
Activez le Mode Développeur (voir méthode auto — étape 1)
Même procédure que ci-dessus.
2
Installez Tizen Studio sur votre PC
Téléchargez depuis :
developer.samsung.com/tizen/download
3
Connectez la TV dans Device Manager
Tizen Studio → Device Manager → Remote Device Manager → ajoutez l'IP de la TV → Connectez.
4
Installez le .wgt
Téléchargez le .wgt ci-dessous, puis dans Tizen Studio : clic droit sur la TV → Install Widget → sélectionnez le fichier.
⬇ Télécharger PIPSILY-TV.wgt
⚠ Méthode avancée pour développeurs. Préférez l'installateur automatique.

Apple TV
Diffusez PIPSILY sur Apple TV via AirPlay depuis votre iPhone ou iPad. Aucune installation sur la TV.
✓ Apple TV HD (4e gen)
✓ Apple TV 4K (1re, 2e, 3e gen)
✓ iPhone / iPad avec iOS 12+
1
Installez PIPSILY sur votre iPhone/iPad
Suivez d'abord le guide iPhone / iPad (onglet 🍎) pour ajouter PIPSILY à votre écran d'accueil.
2
Même réseau Wi-Fi
iPhone/iPad et Apple TV doivent être sur le même réseau Wi-Fi.
3
Option A — Miroir d'écran complet
Sur iPhone : Centre de contrôle (glissez depuis le coin haut droit) → Miroir d'écran → sélectionnez votre Apple TV.
Toute l'interface PIPSILY s'affiche sur la TV.
4
Option B — AirPlay vidéo uniquement
Dans le lecteur PIPSILY, appuyez sur l'icône AirPlay (📡 rectangle + triangle) → sélectionnez votre Apple TV.
La vidéo bascule sur la TV en qualité maximale. Votre iPhone devient télécommande.
5
Posez votre téléphone
Une fois la lecture lancée, vous pouvez poser votre iPhone — la diffusion continue indépendamment.
💡 Recommandation : Utilisez le Miroir d'écran pour naviguer dans le catalogue depuis la TV. Utilisez AirPlay vidéo pour la meilleure qualité en lecture.
⚠ AirPlay absent ? Vérifiez que les deux appareils sont sur le même Wi-Fi et que AirPlay est activé sur l'Apple TV (Paramètres → AirPlay et HomeKit).
