Ce qu'il te faut

📺Samsung Smart TV 2016 ou plus récente (Tizen)
📶TV et ordinateur sur le même réseau Wi-Fi
🪟Windows 10 / 11 ou macOS 12+
⏱Environ 5 minutes pour l'installation

Comment installer

1

Activer le mode développeur sur la TV

  • Télécommande → Accueil → Paramètres ⚙
  • → Support → À propos de ce TV
  • Cliquer 5 fois sur le numéro de modèle affiché
  • Un dialogue apparaît → Mode développeur → ON
  • Entrer l'adresse IP de ton PC/Mac (l'installateur te l'indique)
  • Confirmer → Redémarrer la TV
💡

L'IP de ta TV se trouve dans : Paramètres → Général → Réseau → État du réseau → Informations IP

2

Télécharger et extraire le ZIP

  • Clique sur le bouton de téléchargement correspondant à ton système
  • Windows : clic droit sur le ZIP → Extraire tout
  • Mac : double-clic sur le ZIP (extraction automatique)
3

Lancer l'installateur et suivre les étapes

  • Windows : double-clic sur INSTALLER.bat
  • Mac : double-clic sur INSTALLER.command
    Si macOS bloque → clic droit → Ouvrir → Ouvrir
  • L'installateur connecte la TV, installe l'app et la lance automatiquement

Questions fréquentes

La TV ne se connecte pas
Vérifie que :
• Le mode développeur est bien activé et que la TV a redémarré
• L'adresse IP du PC/Mac est correctement entrée dans les paramètres de la TV
• La TV et l'ordinateur sont sur le même réseau Wi-Fi (pas un réseau invité)
• Le pare-feu Windows n'est pas en train de bloquer la connexion
macOS dit "fichier non sécurisé"
C'est normal pour les scripts non signés par l'App Store. Solution :
Clic droit sur INSTALLER.command → Ouvrir → Ouvrir
Ou dans Terminal : chmod +x INSTALLER.command
Je ne trouve pas mon numéro de modèle sur la TV
Va dans Paramètres → Support → À propos de ce TV.
Le numéro de modèle est affiché à l'écran (ex: QN55Q80C). Clique dessus 5 fois avec OK sur la télécommande.
L'app se met-elle à jour automatiquement ?
Oui. À chaque lancement, PIPSILY TV vérifie si une mise à jour est disponible.
Si c'est le cas, une notification apparaît et l'installation se fait en un clic, sans refaire la procédure d'installation.
Comment désinstaller l'application ?
Sur la TV : Accueil → Applications → (icône paramètres ⚙) → PIPSILY TV → Supprimer
Ma TV Samsung est compatible ?
Toutes les Samsung Smart TV depuis 2016 (système Tizen 3.0+) sont compatibles.
Modèles : Séries F, H, J, K, M, N, R, Q, et plus récents.
